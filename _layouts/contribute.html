Frequently Asked Questions

Common questions about contributing to our archive.

How do I submit a new airdrop project?

You can submit new airdrop projects through our GitHub repository. Fork the repository, create a new markdown file with the project details, and submit a pull request. We'll review and merge valid submissions.

What information should I include when contributing?

Include project name, description, eligibility criteria, token distribution details, timeline, and any relevant links. The more comprehensive the information, the better it helps our community.

How can I report incorrect information?

You can report incorrect information by creating an issue on our GitHub repository or contacting us directly. Please provide the correct information and sources when possible.

Can I contribute if I'm not a developer?

Absolutely! We welcome contributions from everyone. You can help by submitting new airdrop information, reporting bugs, suggesting features, or helping with documentation and community support.
